* Swift 5.0 +
* iOS 11.0 +

## Installation through [CocoaPods](https://github.com/CocoaPods/CocoaPods)

```ruby
use_frameworks!

target '$MySuperApp$' do
    pod 'AKWelcomeKit'
end
```
